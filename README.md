# HomeInventoryManager - 家庭物品库存管理应用

## 项目简介

HomeInventoryManager 是一款专为家庭场景设计的物品库存管理应用，旨在帮助用户轻松梳理、追踪家中各类物品的信息，解决家庭物品杂乱、查找困难、遗忘存放位置等问题。通过简洁直观的操作界面，用户可快速记录物品详情、查询物品信息、管理物品分类，让家庭物品管理更高效、有序。

本应用基于 Android 平台开发，采用原生开发技术栈，注重操作流畅性和用户体验，适配主流 Android 设备，满足不同家庭用户的使用需求。

## 已实现功能

经过开发迭代，目前应用已完成核心基础功能的开发与优化，主要包括以下模块：

### 1. 应用基础架构搭建

- 完成 Android 项目基础配置，优化 AndroidManifest.xml 清单文件，解决应用闪退问题，保障应用稳定启动与运行；

- 搭建清晰的页面架构，包含 MainActivity（主页面）、ItemEditActivity（物品编辑页面）、QueryItemActivity（物品查询页面）、SettingActivity（设置页面）四大核心页面，实现页面间的正常跳转与数据交互；

- 配置应用主题样式、图标资源，保证界面简洁美观，提升用户视觉体验；

- 申请并配置必要权限（振动权限、外部存储读写权限），适配 Android 10+ 存储访问规则，保障功能正常运行的同时兼顾系统兼容性。

### 2. 物品信息编辑功能

- 支持用户手动录入物品核心信息，包括物品名称、类别（如电器、家具、日用品、食品等）、购买时间、存放位置、保质期（可选，针对食品、药品等）、备注说明等；

- 提供表单校验功能，避免关键信息缺失（如物品名称为空），提升数据录入的规范性；

- 支持对已录入的物品信息进行修改、删除操作，方便用户及时更新物品状态。

### 3. 物品查询功能

- 实现基础的物品查询功能，用户可通过输入物品名称关键词快速检索相关物品；

- 支持按物品类别筛选查询，帮助用户快速定位某一类别的所有物品；

- 查询结果以列表形式展示，清晰呈现物品名称、类别、存放位置等关键信息，点击列表项可查看物品完整详情。

### 4. 基础设置功能

- 支持应用基础设置，如修改应用显示名称、切换主题样式（浅色/深色模式）；

- 提供权限管理入口，用户可查看并调整应用已获取的权限（如存储权限）；

- 支持清除应用缓存功能，帮助用户释放设备存储空间。

### 5. 其他辅助功能

- 振动反馈：在点击按钮、提交表单等操作时提供振动反馈，提升操作手感；

- 数据本地存储：物品信息默认存储在本地设备，保障用户数据隐私安全；

- 界面适配：支持不同屏幕尺寸的 Android 设备，保证在手机、平板等设备上均能正常显示和操作。

## 未来规划功能

为进一步提升应用的实用性和用户体验，后续将重点推进以下功能的开发与优化，分阶段实现应用的全面升级：

### 1. 数据管理优化

- 云同步功能：接入云存储服务（如 Firebase、阿里云等），实现物品数据的云端备份与多设备同步，避免本地数据丢失，支持用户在不同设备上查看和管理物品信息；

- 数据导出/导入：支持将物品数据导出为 Excel、CSV 等格式文件，也可从本地文件导入数据，方便用户进行数据备份和二次编辑；

- 数据统计分析：新增数据统计模块，展示物品类别分布、新增物品趋势等信息，以图表（饼图、柱状图）形式直观呈现，帮助用户了解家庭物品构成。

### 2. 物品管理功能升级

- 物品图片上传：支持拍摄物品照片或从本地相册选择图片，关联至对应物品信息，方便用户通过图片快速识别物品；

- 保质期提醒：针对食品、药品等有保质期的物品，支持设置保质期到期提醒，通过通知栏推送提醒消息，避免物品过期浪费；

- 物品借用管理：新增物品借用记录功能，可记录借用对象、借用时间、归还时间等信息，支持借用到期提醒，方便用户管理借出物品；

- 自定义物品类别：允许用户根据自身需求新增、修改、删除物品类别，提升分类的灵活性和个性化。

### 3. 查询与搜索功能优化

- 多条件组合查询：支持同时按名称、类别、存放位置、购买时间等多个条件组合查询，提升查询精准度；

- 模糊查询与联想提示：优化搜索算法，支持模糊匹配物品名称，同时在输入过程中提供联想提示，提升查询效率；

- 查询历史记录：保存用户的查询历史，方便用户快速重复查询常用物品。

### 4. 界面与交互体验提升

- 主题个性化：提供更多主题风格选择（如简约、温馨、商务等），支持自定义主题颜色，满足不同用户的审美需求；

- 手势操作支持：新增手势操作（如左右滑动切换页面、长按删除物品、下拉刷新列表等），提升操作便捷性；

- 动画效果优化：在页面跳转、弹窗显示、列表加载等场景添加流畅的动画效果，提升应用质感。

### 5. 其他拓展功能

- 家庭共享功能：支持创建家庭群组，邀请家庭成员加入，实现物品信息共享和协同管理，方便家庭共同维护物品库存；

- 扫码识别功能：支持通过扫码快速添加物品（如商品条形码），自动获取物品基础信息，减少手动录入工作量；

- 离线模式优化：增强离线使用体验，支持在无网络环境下正常查看和编辑本地数据，网络恢复后自动同步至云端；

- 性能优化：优化应用启动速度、数据加载效率，减少内存占用，提升应用在低配置设备上的运行流畅性。

## 运行环境要求

- 操作系统：Android 8.0（API 级别 26）及以上版本；

- 存储空间：至少 100MB 可用存储空间（用于安装应用及存储本地数据）；

- 权限要求：振动权限、外部存储读写权限（部分功能需开启）；

- 开发环境：Android Studio 4.2 及以上版本，Java/Kotlin 开发环境。

## 安装与使用说明

1. 将应用安装包（.apk 文件）拷贝至 Android 设备，或通过 Android Studio 直接运行项目至测试设备；

2. 首次启动应用时，根据提示授予必要权限（振动、存储权限）；

3. 进入主页面后，可通过底部导航或按钮进入对应功能页面：
- 点击“添加物品”进入 ItemEditActivity 录入物品信息；

- 点击“查询物品”进入 QueryItemActivity 检索物品；

- 点击“设置”进入 SettingActivity 调整应用配置；
4. 按照表单提示完成信息录入、查询筛选等操作，即可正常使用应用核心功能。
